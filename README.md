## ğŸ¯ Game Mechanics

**Stacking Logic:**
1. Block moves horizontally
2. Click to drop
3. Only overlapping area stays
4. Rest falls with physics
5. Perfect stack = bonus points!

**Scoring:**
- +1 point per successful stack
- +2 bonus for perfect placement
- High score saved locally
